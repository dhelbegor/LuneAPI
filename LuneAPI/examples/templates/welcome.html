Welcome to LuneAPI Template Engine

Features

  • Variable substitution with 
    {{ variable }}
  • Conditionals with 
    {% if condition %} {% endif %}
  • Loops with 
    {% for item in items %} {% endfor %}
  • Includes with 
    {% include template %}
  • Filters with 
    {{ variable|filter }}
  • Comments with 
    {# This is a comment #}
